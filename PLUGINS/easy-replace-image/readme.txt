Replace easily an attachment file by uploading another file or by downloading one from an URL, without deleting the attachment.

== Description ==
This plugin allows you to replace an attachment file by uploading another image or by downloading one from a specified URL, without deleting the attachment. The plugin handles the image sizes generation and the attachment metadata update, and you will see the result right away.

= Demo =
https://youtu.be/7kGf8cz49ZE

== Screenshots ==
1. `Easy Replace Image` general page where you can search for a specific image, select it and then perform the replace of the file either by uploading another one, or by downloading one from a URL of your choise.
2. `Easy Replace Image` direct link for the images in the Media screen, for easy access to the replace feature.
3. `Easy Replace Image` feature on the post pages, under the `Featured image` that is selected.
4. `Easy Replace Image` feature on the attachment page.
